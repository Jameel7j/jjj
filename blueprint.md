# **App Name**: Al-Khair Cafe App

## Core Features:

- User Authentication: Enable users to create accounts and log in via phone number.
- Browse Menu: Display categorized menus of available juices and sandwiches with size and add-on options.
- Real-Time Location Tracking: Allow customers to share their location for precise delivery, improving order delivery speed.  
- Order Submission: Link 'Complete Order' button to automatically send order details, sizes, prices, and customer location to +967730528609 via WhatsApp.
- Payment System Integration: Facilitate secure payment options including cash on delivery and integration with existing Kareem money point of sale (POS) with the ID 1299834.
- Admin Dashboard: Provide a browser and mobile accessible dashboard for the owner to manage products, prices, images (upload directly from phone), banners, content, and store status (open/closed).

## Style Guidelines:

- Primary color: Saturated green (#32CD32) to evoke freshness and natural flavors, referencing the fresh juices offered.
- Background color: Very light green (#F0FFF0), providing a clean and natural backdrop to ensure the product images pop.
- Accent color: Yellow (#FFD700), to highlight key interactive elements like call-to-action buttons.
- Body font: 'PT Sans' for body text for its readability and modern feel. Headline Font: 'Playfair' for its modern serif design, which gives the design a high-end feel.
- Use icons related to juices and sandwiches, with a hand-drawn style.
- Prioritize a clean, card-based layout to showcase menu items effectively. The grid-based layout would work well on multiple devices.
- Incorporate subtle transitions and animations on UI elements during the state transitions.
